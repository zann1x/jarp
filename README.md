# jarp

This is Just Another Rendering Project. The whole purpose of it is to understand vulkan, rendering and all that beautiful stuff a little better and see where we're ending at.

## Supported Platforms

- Windows 64-bit

## Dependencies
### General

- glm
- SDL2
- spdlog
- stb
- tinyobjloader
- Vulkan SDK

### Windows

- SDL2
